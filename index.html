<h1 id="getting-started">Getting Started</h1> <p>This is the documentation for Setu&#39;s <a href="https://www.npmjs.com/package/@setu/upi-deep-links">NPM UPI Deep Links</a> module.</p> <p>It is used to interact with Setu&#39;s <a href="https://docs.setu.co/collect/api#tag/upi-deep-links-overview">UPI Deep Link</a> APIs efficiently. It acts as a wrapper has predictable resource-oriented naming convention; it returns JSON-encoded responses with HTTP response codes, and response body.</p> <h2 id="installation">Installation</h2> <p>This package can be installed like -</p> <pre><code class="language-javascript">npm i @setu/upi-deep-links</code></pre> <p>or by doing this</p> <pre><code class="language-javascript">yarn add @setu/upi-deep-links</code></pre> <h2 id="configuration">Configuration</h2> <p>Once installed, you can instantiate the module by running</p> <pre><code class="language-javascript">const SetuUPIDeepLink = require(&#39;@setu/upi-deep-links&#39;); let setu = new SetuUPIDeepLink({ schemeId: &#39;YOUR SCHEME ID&#39;, jwtSecret: &#39;YOUR JWT SECRET&#39;, setuProductInstanceId: &#39;YOUR PRODUCT INSTANCE ID&#39;, mode: &quot;PRODUCTION&quot; or &quot;SANDBOX&quot; // default - &quot;SANDBOX&quot; });</code></pre> <p>After configuration, refer to <a href="/docs/usage">usage</a> to know more about the underlying methods and how to implement it.</p> <blockquote> <p>If you do not have your <code>schemeId</code> or <code>jwtSecret</code> or <code>setuProductInstanceId</code> please contact <a href="https://support.setu.co/support/home">support</a>.</p> </blockquote>
