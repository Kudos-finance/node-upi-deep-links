Usage

Please note that this module exposes required methods and abstracts certain APIs. If you need any help feel free to react out to us or create an issue

displaySetValues

This method is used to retreive the schemeId, jwtSecret, and setuProductInstanceId set by you while instantiating the module. These values are required for you to generate the UPI Deep Links.

Usage

async () => {
    setu.displaySetValues();
};

Response

{
    "schemeId": "YOUR SCHEME ID",
    "jwtSecret": "YOUR JWT SECRET",
    "setuProductInstanceId": "YOUR PRODUCT INSTANCE ID"
}

createPaymentLink

This method is used to generate the UPI Deep Link retreive the schemeId, jwtSecret, and setuProductInstanceId set by you while instantiating the module. These values are required for you to generate the UPI Deep Links.

Usage

async () => {
    let data = await setu.createPaymentLink({
        dueDate: UTC Timetstamp,
        expiryDate: UTC Timetstamp,
        amountValue: Integer,
        amountExactness: String,
        billerBillID: String,
        payeeName: String,
        validationRules: {
            amount: {
                maximum: Integer,
                minimum: Integer
            }
        }
    });
};

Response

{
    "status": 200 || 401 || 500,
    "success": true || false,
    "data": {
        "name": "payeeName",
        "paymentLink": {
            "shortURL": "SHORT URL",
            "upiID": "UPI ID",
            "upiLink": "UPI LINK"
        },
        "platformBillID": "PLATFORM BILL ID"
    }
}
